# ocean-analysis
##**Opis projektu**
Aplikacja internetowa umożliwia interaktywne wyświetlanie wyników analizy zmian parametrów oceanicznych dla siedmiu wybranych obszarów badawczych. Użytkownik może przeglądać wyniki analiz dotyczących temperatury powierzchni morza (SST), anomalii zasolenia powierzchni morza (SSSA) oraz anomalii wysokości powierzchni morza (SSHA).
